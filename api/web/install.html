Download APK

Installation guide

  1. Download the APK file using the button above
  2. Open the downloaded file — Android may ask for permission
  3. Go to Settings > Security (or Settings > Apps > Special access)
  4. Enable "Install unknown apps" for your browser
  5. Open the APK again and tap Install
  6. Once installed, open Xpylon Connect and sign in with your phone number

How TestFlight works

  1. Install TestFlight from the App Store (free)
  2. Click the invite link we'll send you
  3. TestFlight will open and show Xpylon Connect
  4. Tap Install — the app appears on your home screen
  5. Updates are automatic through TestFlight
